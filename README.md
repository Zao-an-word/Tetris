# 俄罗斯方块游戏（C++ EasyX 实现）
基于 C++ 和 EasyX 图形库开发的经典俄罗斯方块小游戏，支持方块旋转/移动、速度调节、消行加分、深度判定、下一方块预览等核心功能，适配 Windows 平台（Visual Studio/Dev-C++）。

## 📋 项目介绍
### 核心功能
- 方块操作：W 键旋转、A/D 键左右移动、S 键长按加速下落
- 速度调节：E/Q 键永久调速、R 键重置速度、S 键临时加速
- 着陆缓冲：方块触底后 300ms 内仍可左右移动，提升操作容错
- 消行机制：自动检测满行并消除，同步加分、提速、补偿深度
- 预览功能：右侧实时显示下一个即将出现的方块
- 深度判定：方块堆至顶部（深度≤0）触发游戏结束
- 界面展示：网格背景、围墙、实时分数/消行数/深度/速度显示

### 技术栈
- 编程语言：C++（C++11 及以上）
- 图形库：EasyX Graphics Library（Windows 专属）
- 开发环境：Visual Studio 2022 / Dev-C++（推荐 VS2022）

## 🛠️ 环境配置
### 前置条件
1. 操作系统：Windows 7/10/11（EasyX 仅支持 Windows）
2. 编译器：Visual Studio 2022（MSVC）/ Dev-C++（MinGW）
3. EasyX 图形库：
   - 访问 [EasyX 官网](https://easyx.cn) 下载对应编译器版本的安装包
   - 运行安装包，自动完成库文件配置（VS 需管理员权限）

### 快速配置（Visual Studio）
1. 打开 Visual Studio 2022，新建「空项目」；
2. 将核心代码文件（`tetris.cpp`）添加到项目中；
3. 项目属性设置：将「字符集」改为「使用 Unicode 字符集」；
4. 点击「本地 Windows 调试器」编译运行（无需额外配置）。

### 快速配置（Dev-C++）
1. 打开 Dev-C++，新建「C++ 项目」；
2. 将核心代码粘贴到项目文件中，保存为 `tetris.cpp`；
3. 工具 → 编译选项 → 勾选「支持 C++11 标准」；
4. 直接点击「运行」按钮编译执行。

## 🎮 运行说明
### 操作指南
| 按键 | 功能 |
|------|------|
| W / w | 方块顺时针旋转 |
| A / a | 方块向左移动 |
| D / d | 方块向右移动 |
| S / s | 长按临时加速下落（50ms/步），松开恢复原速度 |
| E / e | 永久加快速度（-50ms/步，最低 50ms/步） |
| Q / q | 永久减慢速度（+50ms/步，最高 500ms/步） |
| R / r | 重置速度为初始值（300ms/步） |
| 任意键（游戏结束后） | 退出游戏 |

### 资源文件（可选）
若需自定义围墙纹理，需在项目目录创建 `resources` 文件夹，放入以下文件：
- `围墙.jpg`：围墙纹理图（建议尺寸 20×20 像素）
> 若无需图片，代码已默认使用灰色纯色绘制围墙，可直接运行。

### 运行效果
- 游戏窗口尺寸：780×780 像素
- 初始下落速度：300ms/步（可通过 E/Q/R 键调节）
- 分数规则：每消除一行 +100 分，消行后基础速度 -5ms
- 深度规则：初始深度 37，方块落地更新最小深度，消行 +1 深度补偿

## 📂 代码结构
tetris.cpp
- ├── 全局常量/变量：方块尺寸、游戏区域边界、深度/分数/速度等核心变量
- ├── 核心函数：
- │ ├── drawUnitBlock()：绘制单个带颜色的方块单元（含黑色边框）
- │ ├── drawTetrisBlock()：绘制完整方块（区分游戏区/预览区，预览区跳过边界校验）
- │ ├── bkuni()：绘制围墙单元（图片/纯色矩形容错）
- │ ├── drawbk()：绘制背景、网格、围墙、提示文字和已落地方块
- │ ├── cangoleft/cangoright/canfall/canrotate()：碰撞检测（防越界/防重叠）
- │ ├── moveblock()：处理方块移动/旋转逻辑，成功则刷新画面
- │ ├── automove()：方块自动下落逻辑
- │ ├── willgen()：生成下一方块参数（类型/旋转/位置/颜色）
- │ ├── w_dis()：绘制预览区下一方块
- │ ├── savefell()：保存落地的方块坐标/颜色，更新深度值
- │ ├── judge()：检测方块落地，处理 300ms 着陆缓冲逻辑
- │ ├── remove()：检测并消除满行，更新分数/速度/深度
- │ ├── check_key_press()：检测按键（长按/单次按键）
- │ ├── gameover()：绘制半透明游戏结束界面，展示统计信息
- └── main()：游戏主循环（初始化、按键检测、自动下落、预览绘制、落地判定、画面刷新）

## 🐞 常见问题
### 1. 编译报错“找不到 graphics.h”
- 原因：未安装 EasyX 图形库，或安装版本与编译器不匹配；
- 解决：重新下载对应编译器版本的 EasyX，运行安装包修复。

### 2. 预览区方块不显示
- 原因：预览区坐标被游戏区边界校验过滤；
- 解决：代码中 `drawTetrisBlock()` 已通过 `is_preview` 参数区分预览区，无需额外修改。

### 3. 游戏异常结束（深度突然变为 0）
- 原因：方块坐标越界导致深度计算异常；
- 解决：`savefell()` 仅在方块有效放置时更新深度，且强制 `depth ≥ 0`，无需额外修改。

### 4. 宽字符报错（如 swprintf_s/drawtext）
- 原因：项目字符集未设置为 Unicode；
- 解决：VS 项目属性 → 高级 → 字符集 → 改为「使用 Unicode 字符集」。

### 5. Git 提交代码后编译失败
- 原因：换行符/编码格式问题；
- 解决：
  - 方案1：Git 配置换行符：`git config --global core.autocrlf true`；
  - 方案2：用 VS/Dev-C++ 重新保存代码为 UTF-8 编码。

6. 一键下落：添加空格按键，让方块直接落到底部；
7. 排行榜：展示本地历史最高分，支持清空排行榜。
